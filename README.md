PalindromeChecker App

PalindromeChecker App is a simple Java console application that starts by displaying a welcome message and basic application information. It demonstrates the fundamental structure of a Java program, including the use of classes, the main() method as the entry point, and console output for user interaction.

When the program starts, the Java Virtual Machine (JVM) invokes the main() method, which prints the application name, version, and a startup message to the console. This initial step defines the application's startup behavior before proceeding to further functionality such as palindrome checking.

The project focuses on core Java concepts including:

Class structure as a container for application logic

Main method (public static void main(String[] args)) as the program entry point

Static keyword allowing the JVM to call the main method without creating an object

Console output using System.out.println() to display information

Application flow control for startup execution

This implementation uses a console-based approach and does not require any data structures at the startup stage. It serves as the foundation for building additional features like accepting user input and checking whether a string or number is a palindrome.# PalindromeCheckerApp
